Client-side form validation

If I want the user to have a good experience I must use form validation, a great advantage of doing so is that when incorrect data is found the user can quickly fix it.

Validation done in the browser is called client-side validation

Examples of form validation feedback:

  • Please enter your phone number in the format xxx-xxxx" (A specific data format is required for it to be considered valid).
  • Please enter a valid email address" (the data you entered is not in the right format).

I can validate user data without JavaScript, this is done by using validation attributes on form elements. For example: required, minlength and maxlength. This is called Built-in form validation
