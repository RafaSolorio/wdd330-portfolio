Forms

Tip to make the forms more user-friendly: use the placeholder attribute, it is useful to give the user hints to fill out the form.

Study the list of attributes that can be used to make forms more user-friendly.

It is important to use the correct input field, for example, use a password input field instead of a text input field if I want the user to enter a password.

OOP

There are many benefits to object-oriented programming, for example, using the same functionality for multiple objects, or inheriting attributes and characteristics from one object to a new object.

Remember that names for class declarations are capitalized.

It is recommended to use arrow functions in callbacks because they do not have their own this context.

Modules

There are many advantages using modules, they make the code more organized, and it makes it easier to obtain the functionalities that we want for our program.

The code in a module should have a single purpose.
