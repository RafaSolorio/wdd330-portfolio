Notes:

Transforms don't work on inline elements. To handle it I need to add display: inline-block; to the element

Don't declare a new transform because a second transform would override the first. To declare multiple transformations, use a space-separated list of transform functions.
For example: transform: translateX(40px) scale(1.5);

Scaling and translation have no impact on the document flow.
