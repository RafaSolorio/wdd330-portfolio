Promises

A promise represents the future result of an asynchronous operation.

I think promises are a useful tool because they help simplify the process in a program

The phases of a promise:

  • Pending phase (waiting the results of an operation).
  • Then the promise is settled.
  • Then if the promise has been settled it may have been resolved or rejected
